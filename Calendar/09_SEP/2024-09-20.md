# Delete from Git
1. Be in the local repository
```bash
cd path/to/your/repository
```
2. Remove using `rm`
```bash
git rm --cached path/to/your/file.txt
```
3. Commit changes
```bash
git commit -m "Remove file.txt from repository"
```
4. Push
```bash
git push
```
